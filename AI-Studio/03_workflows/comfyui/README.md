# ComfyUI Workflow Stubs

Place your ComfyUI workflow JSON files here.

Recommended naming:

- `txt2img_base_v1.json`
- `img2img_stylematch_v1.json`
- `upscale_detail_v1.json`

## Suggested Metadata to Track Per Render

- Job ID
- Workflow filename
- Checkpoint/model
- Positive prompt
- Negative prompt
- Seed
- Steps / Sampler / CFG
- Resolution
- Output files

Save metadata logs in `07_reviews/` or `10_logs/` for reproducibility.
